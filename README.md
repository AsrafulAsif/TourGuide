
# Tour Guide

Tour Guide is an an online tour guide platform with interactive features for the consumers. It will provide basic instructions for tour guides. This system will be completely functional, allowing the user to access all relevant information in one place. This system will offer recommendations based on the location provided, and the user will be able to create their own tour criteria lists.
